# ספר המתכונים שלי 🍽️

אפליקציית React לניהול מתכונים, בעברית (RTL), עם חלוקה לקטגוריות, חיפוש, ושינוי גודל מנה אוטומטי.

## הרצה מקומית (לבדיקה במחשב שלך)

1. התקן [Node.js](https://nodejs.org) אם אין לך (גרסה 18 ואילך).
2. פתח טרמינל בתיקיית הפרויקט והרץ:

```bash
npm install
npm run dev
```

3. פתח את הכתובת שתופיע (כמו `http://localhost:5173`) בדפדפן.

## פרסום לאינטרנט (Vercel)

### אם יש לך כבר חשבון Vercel:

**אפשרות א' - דרך GitHub (מומלץ):**
1. צור repository חדש ב-GitHub והעלה את כל הקבצים האלה.
2. ב-Vercel: New Project → ייבא את ה-repository.
3. Vercel יזהה אוטומטית שזה פרויקט Vite ויגדיר הכל. לחץ Deploy.
4. בתוך דקה תקבל כתובת אינטרנט חיה כמו `recipe-book.vercel.app`.

**אפשרות ב' - דרך Vercel CLI (ישיר מהמחשב, בלי GitHub):**
```bash
npm install -g vercel
vercel
```
עקוב אחר ההוראות בטרמינל (התחברות לחשבון, אישור הגדרות). בסוף תקבל כתובת חיה.

### עדכון לאחר שינויים:
- אם חיברת ל-GitHub: כל `git push` יגרום ל-Vercel לבנות ולפרסם גרסה חדשה אוטומטית.
- אם השתמשת ב-CLI: הרץ `vercel --prod` מתוך התיקייה כדי לפרסם גרסה מעודכנת.

## הוספת מתכונים חדשים

המתכונים נמצאים בקובץ `src/App.jsx`, במשתנה `SAMPLE_RECIPES` בתחילת הקובץ.
כדי להוסיף מתכון חדש, שלח לקלוד תמונה של המתכון בצ'אט כמו שעשית עד עכשיו —
קלוד יוסיף את הקוד המתאים, ותצטרך:
1. להחליף את הקובץ `src/App.jsx` בגרסה המעודכנת.
2. לעשות commit + push (אם עובד עם GitHub) או `vercel --prod` (אם עובד עם CLI).
